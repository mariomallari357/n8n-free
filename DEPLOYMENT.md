# Deployment worksheet

This checkout is prepared for a free n8n deployment using:

- GitHub fork owner: `mariomallari357`
- Hugging Face Docker Space: `<hf-owner>/<space-name>`
- Supabase Postgres: use the **Session pooler** connection (`port 5432`) unless
  the Space can reach the project's direct IPv6 endpoint.
- Timezone: `Asia/Shanghai`

## Supabase

Create the project, then run this in the SQL Editor:

```sql
CREATE TABLE "keep-alive" (
  id BIGINT generated BY DEFAULT AS IDENTITY,
  name text NULL,
  created_at timestamptz DEFAULT now(),
  CONSTRAINT "keep-alive_pkey" PRIMARY KEY (id)
);
INSERT INTO "keep-alive"(name) VALUES ('ping');
```

From the **Connect** panel, copy the Session pooler values. Its username is
usually `postgres.<project-ref>`, not simply `postgres`.

## Hugging Face Space secrets

```text
DB_POSTGRESDB_HOST=<Supabase Session pooler host>
DB_POSTGRESDB_PORT=5432
DB_POSTGRESDB_DATABASE=postgres
DB_POSTGRESDB_USER=postgres.<project-ref>
DB_POSTGRESDB_PASSWORD=<database password>
DB_POSTGRESDB_SSL_CA=<Supabase CA certificate PEM>
N8N_ENCRYPTION_KEY=<stable random value; never rotate after storing credentials>
```

## Hugging Face Space variables

Replace `<hf-owner>` and `<space-name>` first.

```text
DB_TYPE=postgresdb
DB_POSTGRESDB_SCHEMA=public
N8N_PORT=7860
N8N_PROTOCOL=https
N8N_HOST=<hf-owner>-<space-name>.hf.space
N8N_EDITOR_BASE_URL=https://<hf-owner>-<space-name>.hf.space
WEBHOOK_URL=https://<hf-owner>-<space-name>.hf.space
N8N_ENFORCE_SETTINGS_FILE_PERMISSIONS=true
N8N_PUSH_BACKEND=websocket
DB_POSTGRESDB_SSL_REJECT_UNAUTHORIZED=true
N8N_PROXY_HOPS=1
GENERIC_TIMEZONE=Asia/Shanghai
TZ=Asia/Shanghai
EXECUTIONS_DATA_SAVE_ON_PROGRESS=false
EXECUTIONS_DATA_SAVE_ON_ERROR=all
EXECUTIONS_DATA_SAVE_ON_SUCCESS=none
EXECUTIONS_DATA_SAVE_MANUAL_EXECUTIONS=false
EXECUTIONS_DATA_PRUNE=true
EXECUTIONS_DATA_MAX_AGE=168
```

## GitHub Actions

Add these repository secrets:

```text
HF_TOKEN=<Hugging Face write token>
SUPABASE_URL=<Supabase project URL>
SUPABASE_KEY=<Supabase anon public key>
```

Add this repository variable:

```text
HF_REPO=<hf-owner>/<space-name>
```

Then manually run the **Push to huggingface** workflow, or push a commit to
`main`. Confirm `https://<hf-owner>-<space-name>.hf.space/healthz/readiness`
returns a successful response before creating the n8n owner account.
